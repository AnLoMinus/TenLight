השילוב בין שני הצדדים יוצר שלמות: אי אפשר לעבוד את ה׳ בלי אהבת הזולת, ואי אפשר לבנות חברה מוסרית ללא עוגן רוחני. הדיברות הן אבני היסוד של ברית נצח בין שמיים לארץ.
